Anatomy of html

  1. The starting tag of the html skeleton must be <!DOCTYPE html> and it must be capital
  2. All tags must have an opening tag < and a closing tag >
  3. All of the main information of a webpage should be in the body tag
  4. All html code should be tabbed and indented for visibility of code
  5. The style tag should be under the head but it can also be in the body
  6. the html tag is always the second tag
  7. the head tag is always the third tag
  8. Doctype is always the first tag
  9. all tags with content inside need to be closed
